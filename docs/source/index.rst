Welcome to the documentation for AstroPT, a project for training astronomical large observation
models. This repository began its life as Andrej Karpathy's
`nanoGPT <https://github.com/karpathy/nanoGPT>`_, and has been altered to be usable
for astronomical observation data.

Check out the `UniverseTBD <https://universetbd.org/>`_ Discord for updates:
`https://discord.gg/MNEVegvfJq <https://discord.gg/MNEVegvfJq>`_
